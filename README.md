## 📈 Project 1 — Sales Analytics Dashboard

### Overview
An interactive Excel dashboard analyzing sales performance across 
categories, states, subcategories, and time periods — tracking 
revenue, profit, quantity, and order trends with Year-over-Year 
growth metrics.

---

### 📊 Key KPIs

| Metric | Value | YoY Growth |
|---|---|---|
| **Total Sales** | $22,97,200.86 | 20.09% ✅ |
| **Total Profit** | $2,86,397.02 | 14.41% ✅ |
| **Total Quantity** | 37,873 | 27.45% ✅ |
| **No of Orders** | 9,994 | 28.64% ✅ |
| **Profit Margin** | 12.47% | 1% ⚠️ |

---

### 📉 Key Visualizations

| Visual | Insight |
|---|---|
| **Sales & Profit Analysis** | Monthly trend with YoY comparison across 2014-2017 |
| **Sales by State** | US map showing state-wise sales distribution |
| **Top 5 Subcategories** | Phones ($330.05K), Chairs ($328.45K), Storage ($223.84K), Tables ($206.91K), Binders ($203.41K) |
| **Category Wise Profit** | Technology vs Office Supplies vs Furniture |
| **Category Wise Sales Share %** | Donut chart showing category distribution |
| **Interactive Filters** | Year, Region (Central, East, South, West), Segment (Consumer, Corporate, Home Office) |

---

### 🔑 Key Insights

| Finding | Detail |
|---|---|
| **Top Subcategory** | Phones lead with $330.05K in sales |
| **Most Profitable Category** | Technology highest profit margin |
| **Best Growth Metric** | Orders grew 28.64% YoY |
| **Profit Margin Concern** | Only 1% YoY growth — needs attention |
| **Regional Coverage** | All 4 US regions tracked — Central, East, South, West |

---

### ⚙️ Excel Features Used

| Feature | Where Used |
|---|---|
| **Pivot Tables** | Sales aggregation by category, state, subcategory |
| **Line Charts** | Monthly sales and profit trend analysis |
| **Bar Charts** | Top 5 subcategories comparison |
| **Donut Chart** | Category wise sales share |
| **Map Chart** | State wise sales visualization |
| **Slicers** | Year, Region, Segment interactive filters |
| **KPI Cards** | Sales, Profit, Quantity, Orders, Margin |
| **Conditional Formatting** | YoY growth positive/negative indicators |

## 👥 Project 2 — HR Analytics Dashboard

### Overview
A comprehensive Excel HR analytics dashboard tracking employee 
attrition patterns, demographic breakdowns, department trends, 
education impact, and job satisfaction metrics — helping HR teams 
identify retention risks and workforce patterns.

---

### 📊 Key KPIs

| Metric | Value |
|---|---|
| **Total Employees** | 1,470 |
| **Attrition Count** | 237 |
| **Active Employees** | 1,233 |
| **Attrition Rate** | 16.12% |
| **Average Age** | 37 |
| **Job Satisfaction Rating** | 2.6/5 ⭐ |

---

### 📉 Key Visualizations

| Visual | Insight |
|---|---|
| **Employees by Gender** | Female: 588, Male: 882 |
| **Education Wise Attrition** | Bachelor's (99), Master's (58), Associate's (44), High School (31), Doctoral (5) |
| **Attrition by Job Role** | Sales Representatives highest attrition |
| **Department Wise Attrition** | R&D (56.12%), Sales (38.82%), HR (5.06%) |
| **Attrition by Age Group** | 25-34 highest (112), 35-44 (51), Under 25 (38), 45-54 (25), Over 55 (11) |
| **Attrition by Marital Status** | Married (473), Single (470), Divorced (327) |
| **Interactive Filter Panel** | Education Field, Department, Job Role filters |

---

### 🔑 Key Insights

| Finding | Detail |
|---|---|
| **Highest Attrition Dept** | R&D — 56.12% of total attrition |
| **Most Affected Age Group** | 25-34 years — 112 employees left |
| **Gender Distribution** | 60% Male, 40% Female workforce |
| **Education Impact** | Bachelor's degree holders show highest attrition |
| **Job Satisfaction** | Low at 2.6/5 — key retention concern |
| **Overall Attrition Rate** | 16.12% — above healthy 10% benchmark |
| **Active Workforce** | 1,233 out of 1,470 total employees |

---

### ⚙️ Excel Features Used

| Feature | Where Used |
|---|---|
| **Pivot Tables** | Attrition aggregation by dept, age, education |
| **Bar Charts** | Education wise and job role attrition |
| **Donut Chart** | Department wise attrition share |
| **Column Charts** | Age group attrition comparison |
| **KPI Cards** | Total employees, attrition, active count |
| **Slicers** | Education field, department, job role filters |
| **Conditional Formatting** | Attrition rate highlights |
| **Star Rating Visual** | Job satisfaction display |
